A full-stack web application that allows users to explore, add, update, and manage drama recommendations.

#Live Demo
GitHub Pages: https://allie123413.github.io/Drama/
Local Server (Node.js): http://localhost:3000

#Features
- View drama list
- Add new drama
- Edit existing dramas
- Delete dramas
- View detailed drama information
- About Me page with top recommendations

Tech Stack
- Node.js
- Express.js
HTML / CSS
- Bootstrap 5
- JavaScript

How to Run Locally
- Install dependencies: npm install express
- Run the server: node server.js
- Open in browser: http://localhost:3000

API Routes
- / → Home page
- /lists → Drama list
- /addDrama → Add drama
- /updateDrama/:id → Edit drama
- /deleteDrama/:id → Delete drama
- /viewDetails/:id → Drama details
- /aboutMe → About page

Learning Outcomes
- CRUD operations (Create, Read, Update, Delete)
- Express routing (GET / POST)
- Bootstrap layout design
- Server-side rendering with Node.js
